workshop month 2017

ANDROID WORKSHOP

IEEE Java and Android Team has a vision to build a great community of Android developers in Benha University having a great potential to help the student to start their learning journey in Java and Android As we offer Java and Android workshops and sessions for student also provide all the contents and courses links which they need. We made a Java workshop in the first term 2016/2017 through 3 weeks in 6 sessions took place in Shoubra Faculty of Engineering. While IEEE Benha University student branch participated in organization of the workshop, as technical support and Instructors. The workshop covered all the basics of Java programming language to get start in Object Oriented Approach with Java then prepare the students to start their learning in Android development after workshop.

PYTHON WORKSHOP

We strongly believe that the programming is an essential tool required in most fields ,our aim is not only to educate school students in their early stage of their joining school but also the under graduates .For this aim ,we set up two workshops , the first one held in Shoubra Faculty of Engineering , and the second one in CIC university . The registration was open for all, our faculty students and other faculties’ students . Regarding the first one that held in Shoubra , it was initiated by an opening session during which DR.Islam al sharawy (PHD in computer science ) delivered a speech addressing programming applications as a tool to trigger people to enhance mental capacity and as a way to build a career .the four day –workshop contain all the basics of python programming language . Following this workshop, many students start thinking whether shifting their careers or utilizing the programming in their major fields.
